# Probably True · The Notebook Surface

Welcome to the **derivation half** of Probably True.

If you arrived from the [interactive chapters](https://probably-true.vercel.app/),
you've already seen the visual intuition — the dragging, the sliders, the "ohh"
moments. This surface shows the **other half**: the math, the code, the proofs.

Each chapter here is a Jupyter notebook with:

- 📐 **Math derivations** in LaTeX — every formula proven from first principles
- 🐍 **Python code** using `numpy`, `matplotlib`, `scipy`, fully visible
- 🎬 **Animations** — generated with `matplotlib` + `celluloid`, watch the math unfold
- 📥 **Downloadable** — each chapter has an `.ipynb` button so you can run it locally
  or in [Google Colab](https://colab.research.google.com/)

## Why two surfaces?

The interactive chapters answer **"what does this *feel* like?"**
The notebooks answer **"what's actually *happening* mathematically?"**

You don't need both. Pick whichever one lands for the way you learn.

## How to use these notebooks

**Read in browser:** the rendered HTML is enough for understanding.

**Run yourself:** click the download button (top right of any chapter) for the
`.ipynb` file, then open it in Jupyter Lab, VS Code, or upload to Google Colab.

**Modify and explore:** every code cell is editable when you run it locally.
Change a parameter, re-run, see what changes. That's where the deepest learning
happens.
